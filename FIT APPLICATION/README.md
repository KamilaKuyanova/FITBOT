# FitApplication (Expo mobile)

Mobile app lives here:

`FIT APPLICATION/mobile`

## Quick start
```bash
git clone https://hub.quadcode.ai/QuadcodeAI/fitapplication.git
cd "fitapplication/FIT APPLICATION/mobile"
npm install
npx expo start --tunnel
```

## Open on phone
- Install Expo Go
- Scan the QR code from the terminal/browser

## Optional: Web support
If web bundling fails due to react-native-web:
```bash
npm install react-native-web react-dom
```

## Troubleshooting
**Clone auth failed**
Repo is private. You need access or a Personal Access Token (PAT) as your password.

**Expo version mismatch**
```bash
npx expo install expo@~54.0.30 babel-preset-expo@~54.0.9
```
